Allow users to dynamically adjust the Search Limit Budget from the UI (e.g., a slider or dropdown selecting from 1 to 8 searches) instead of hardcoding it to 4. This value would be passed to the backend API.

Copy to Clipboard: A clean floating button on the Research Summary card to instantly copy the generated markdown.
Download as PDF/Markdown: Export the summary and the query logs into a neat .md file for offline reading.
3. 🌐 Web Links & Favicons in Sources
Currently, the source results are shown as plain text. We can:

Parse URLs from the DuckDuckGo search outputs and display clickable hyperlinked sources.
Display the website's favicon (e.g., Google, Wikipedia) next to each source snippet.
4. 📜 Search History Sidebar
Keep a history of previous research queries in a collapsible left sidebar (using localStorage so they persist when the page is reloaded). Users can click on a past query to instantly load its final summary without re-running the search.

Add a floating theme controller to switch between the premium Neon Glassmorphic Dark Mode (current) and a clean, high-contrast Light Mode suitable for long-form reading.
